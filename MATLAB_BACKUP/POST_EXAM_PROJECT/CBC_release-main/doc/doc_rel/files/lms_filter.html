Least Mean Squares (LMS) Filter

The Least Mean Squares (LMS) filter is an adaptive filtering algorithm designed for harmonic estimation. It minimises the error between an input signal and its estimated harmonic reconstruction. This makes it particularly effective for applications requiring real-time signal processing, harmonic analysis, and dynamic signal modelling.

Functionality

The LMS filter operates by continuously updating its internal weight coefficients to match the input signal as closely as possible. It requires a predefined harmonic basis signal, consisting of sine and cosine terms, to estimate the harmonic content of the input.

Inputs

  • Input Signal: The primary signal that the LMS filter is tasked with estimating.
  • Harmonic Basis: A structured signal containing sine and cosine components corresponding to different harmonic frequencies. The filter expects this input to have a fixed size of 1 × 17. If using the provided Harmonic Basis Generator, this requirement is automatically satisfied.

Outputs

  • Estimated Signal: The reconstructed signal based on the LMS filter’s dynamic estimation of harmonic components.
  • Harmonic Coefficients: A set of weights corresponding to the contributions of each harmonic component in the estimated signal. These coefficients indicate the magnitude of the sine and cosine terms at different harmonic frequencies.

Internal Variable

The LMS filter has a single tunable internal variable:

  • lms_var_gain – This controls the adaptation gain of the filter. It defaults to 0.01 but can be adjusted to optimise performance.
    • Lower values (< 0.01) may slow down convergence.
    • Higher values (> 0.2) can lead to instability, causing the output to become NaN. It is advised to keep this value below 0.2 for reliable operation.

Usage Considerations

  • The harmonic basis signal must always have a size of 1 × 17 to ensure compatibility with the LMS filter.
  • The adaptation gain should be tuned carefully to balance convergence speed and stability.
  • The estimated harmonic coefficients can be used for further analysis or signal reconstruction.

Conclusion

The LMS filter is a powerful tool for real-time harmonic estimation and adaptive signal processing. Its ability to dynamically adjust to input variations makes it highly effective for various engineering applications, particularly in signal reconstruction and adaptive filtering tasks.

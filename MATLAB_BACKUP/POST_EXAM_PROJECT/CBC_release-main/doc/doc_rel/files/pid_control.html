PID Controller

The PID Controller provides proportional-integral-derivative (PID) control functionality for real-time systems. It is designed to maintain optimal system performance by dynamically adjusting the control output based on error feedback. The controller includes a built-in filtering mechanism to reduce high-frequency content, improving system stability and robustness.

Functionality

The PID controller continuously computes control corrections using the proportional, integral, and derivative terms. The proportional term reacts to present error, the integral term accounts for accumulated past errors, and the derivative term anticipates future errors by considering the rate of change.

Inputs

  • System Error: The difference between the desired setpoint and the measured system response.

Outputs

  • Control Output: The computed control signal applied to the system to minimise error.

Internal Variables

The PID controller has four tunable internal variables:

  • pid_var_kp – The proportional gain. Determines how strongly the controller reacts to the current error. Defaults to 0.
  • pid_var_kd – The derivative gain. Helps reduce oscillations by damping rapid error changes. Defaults to 0.
  • pid_var_ki – The integral gain. Eliminates steady-state error by accumulating past errors. Defaults to 0.
  • pid_var_Fc – The cutoff frequency (Hz) for the built-in filtering mechanism. Controls the attenuation of high-frequency components in the derivative control. Defaults to 1.

Usage Considerations

  • The proportional, integral, and derivative gains should be tuned carefully to achieve the desired control performance without instability.
  • The filtering mechanism can be adjusted using pid_var_Fc to manage the influence of high-frequency noise.
  • The controller is designed for real-time applications and should be used with an appropriate fixed sampling rate.

Conclusion

The PID controller is a robust and adaptive control solution, well-suited for real-time applications requiring precise and stable system regulation. Its tunable parameters allow for flexible optimisation to meet specific control objectives.

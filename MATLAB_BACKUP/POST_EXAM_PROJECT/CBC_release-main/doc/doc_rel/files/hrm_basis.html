Harmonic Basis Block

The Harmonic Basis block generates a harmonic signal composed of sine and cosine components. This signal serves as a basis for use in adaptive filtering algorithms such as the Least Mean Squares (LMS) and Recursive Least Squares (RLS) filters. The block outputs a structured harmonic waveform based on configurable fundamental frequency and harmonic coefficients.

Functionality

The block computes a vector containing the sine and cosine components of a harmonic signal at predefined harmonic frequencies. Specifically, the output consists of: 

U = [DC, cos(1 × ωt), ..., cos(n × ωt), sin(1 × ωt), ..., sin(n × ωt)]

where ω is the fundamental frequency and t represents time.

Output Signal

The block outputs a fixed-size vector of 1 × 17, consisting of:

  • DC component.
  • 8 cosine components of harmonic frequencies.
  • 8 sine components of harmonic frequencies.

Usage Considerations

The Harmonic Basis block is unprotected and fully configurable, allowing users to adjust the number of harmonics. The generated signal can be used as an input to adaptive filters or other signal processing applications.

Conclusion

The Harmonic Basis block is an essential component for generating structured harmonic signals, making it ideal for applications requiring predefined sine and cosine basis functions.

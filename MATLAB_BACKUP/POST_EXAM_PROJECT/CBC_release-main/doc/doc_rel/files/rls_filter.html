Recursive Least Squares (RLS) Filter

The Recursive Least Squares (RLS) filter is an adaptive filtering algorithm used for precise and rapid harmonic estimation. Unlike the Least Mean Squares (LMS) filter, the RLS filter minimises the estimation error by continuously updating its internal weight coefficients using an exponentially weighted least squares criterion. This makes it highly effective for real-time signal tracking, adaptive modelling, and dynamic system identification.

Functionality

The RLS filter operates by adjusting its internal weights based on incoming data, ensuring that the estimated signal closely matches the input. It requires a harmonic basis input signal that consists of predefined sine and cosine components, enabling accurate harmonic estimation.

Inputs

  • Input Signal: The primary signal that the RLS filter aims to estimate.
  • Harmonic Basis: A structured signal containing sine and cosine components at different harmonic frequencies. The input signal size must be fixed at 1 × 17. The provided Harmonic Basis Generator block ensures compatibility with this requirement.

Outputs

  • Estimated Signal: The reconstructed signal based on the RLS filter’s adaptive estimation of harmonic components.
  • Harmonic Coefficients: A set of adaptive weights corresponding to the contribution of each harmonic component in the estimated signal.

Internal Variables

The RLS filter relies on internal variables for adaptation and control:

  • rls_var_delta – A regularisation parameter that prevents instability in the early iterations. The default value is 1.
  • rls_var_lambda – The forgetting factor that determines the influence of past data on the current estimation. The default value is 0.95. A value close to 1 ensures long-term memory, while lower values emphasise recent data.

Usage Considerations

  • The harmonic basis signal must always be of size 1 × 17 to ensure correct operation.
  • The forgetting factor should be tuned carefully to balance adaptability and stability.
  • The estimated harmonic coefficients can be used for further signal analysis or reconstruction.

Conclusion

The RLS filter is a powerful and efficient tool for real-time adaptive filtering and harmonic estimation. Its rapid convergence and precision make it ideal for applications requiring dynamic system modelling and signal tracking.

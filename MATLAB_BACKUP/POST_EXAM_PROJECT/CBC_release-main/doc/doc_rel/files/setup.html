Setup Guide

Welcome to the setup guide for the MATLAB library. This document will guide you on how to properly set up and use setup_cbc.m to initialise the environment.

Purpose of setup_cbc.m

The setup_cbc.m script is designed to set up the MATLAB environment by:

  • Ensuring the script runs from its own directory.
  • Extracting protected project blocks (.mlproj files) into folders.
  • Adding required directories to the MATLAB path.
  • Preparing the workspace for library usage.

How to Use setup_cbc.m

To use the library, you need to run setup_cbc.m in MATLAB. Follow these steps:

  1. Navigate to the folder containing setup_cbc.m in MATLAB.
  2. Run the script by typing:
    3. setup_cbc
  3. Wait for the script to finish. It will process the required files and set up the environment.

Understanding Project Blocks

The setup script extracts protected blocks from .mlproj files and expands them into accessible directories. These extracted folders are then added to the MATLAB path, ensuring all necessary components are available.

Library Path Configuration

To properly use the library, all necessary files must be available on the MATLAB path. The setup script automatically adds:

  • The src directory and its subfolders.
  • Any extracted project folders.
  • The library directory and its subfolders.

Completion

Once the script has completed successfully, you can start using the library components immediately. The necessary files are now available on the MATLAB path.
